## 关于应用 (About the Application)

### 飞秋通讯是什么？

飞秋通讯是一款基于局域网（LAN）的即时通讯应用，实现了 FeiQ 和 IPMsg 协议，支持点对点的消息传输、文件传输和群聊功能。

**主要特点：**

- 无需互联网连接即可使用
- 无需服务器，点对点通信
- 自动发现局域网内的其他用户
- 跨平台支持（Windows、macOS、Linux）
- 开源免费（MIT 许可证）

### 飞秋通讯与其他通讯应用有什么区别？

| 特性       | 飞秋通讯 | 微信/QQ  | 钉钉/飞书 |
| ---------- | -------- | -------- | --------- |
| 需要互联网 | ❌ 否    | ✅ 是    | ✅ 是     |
| 需要服务器 | ❌ 否    | ✅ 是    | ✅ 是     |
| 局域网通信 | ✅ 是    | ❌ 否    | ❌ 否     |
| 数据隐私   | 完全本地 | 云端存储 | 云端存储  |
| 开源       | ✅ 是    | ❌ 否    | ❌ 否     |

**适用场景：**

- 企业内网环境
- 对数据安全有要求的场合
- 不稳定的网络环境
- 离线环境（如实验室、工厂）

### 飞秋通讯是免费的吗？

是的，飞秋通讯完全免费且开源：

- ✅ 无需支付任何费用
- ✅ 无广告
- ✅ 无用户数量限制
- ✅ 源代码完全开放（GitHub）

### 飞秋通讯需要联网吗？

不需要互联网连接，但需要在同一个局域网内：

**网络要求：**

- 设备必须在同一个局域网（LAN）内
- 可以通过 Wi-Fi 或有线网络连接
- 支持 UDP 广播通信
- 子网掩码允许广播通信（通常为 255.255.255.0）

**注意：** 跨互联网的通信（如不同办公室之间）需要 VPN 支持。

## 兼容性 (Compatibility)

### 飞秋通讯支持哪些操作系统？

飞秋通讯支持以下操作系统：

| 操作系统 | 支持版本                              | 状态        |
| -------- | ------------------------------------- | ----------- |
| Windows  | Windows 10/11                         | ✅ 完全支持 |
| macOS    | macOS 11+ (Big Sur 及以上)            | ✅ 完全支持 |
| Linux    | Ubuntu 20.04+, Fedora 33+, Arch Linux | ✅ 完全支持 |

### 可以与其他 FeiQ/IPMsg 客户端互通吗？

是的，飞秋通讯实现了标准的 IPMsg 协议和 FeiQ 扩展协议：

**已测试兼容的客户端：**

- ✅ IPMsg（官方版本）
- ✅ 飞秋（FeiQ）
- ✅ 蜜蜂（MieMsg）
- ✅ 其它 IPMsg 协议兼容客户端

**协议支持：**

- 标准 IPMsg 协议（基本互通）
- FeiQ 扩展协议（完整功能）

**注意：** 不同客户端的实现可能有细微差异，某些高级功能可能不完全兼容。

### 可以在手机上使用飞秋通讯吗？

目前飞秋通讯仅支持桌面操作系统（Windows、macOS、Linux）。

**原因：**

- 飞秋通讯使用 UDP 广播进行用户发现
- 移动操作系统（iOS/Android）对后台 UDP 广播有限制
- 屏幕尺寸较小，不适合完整的桌面界面

**替代方案：**

- 如果需要在手机上使用，可以尝试其他支持 IPMsg 协议的移动客户端
- 或使用远程桌面连接到运行飞秋通讯的电脑

### 旧版本可以连接新版本吗？

是的，飞秋通讯保持了协议兼容性：

**版本兼容性：**

- 不同版本之间可以正常通信
- 建议所有用户使用最新版本以获得最佳体验
- 新版本可能添加新功能，但不影响基本通信

## 安全性 (Security)

### 飞秋通讯的通信是加密的吗？

飞秋通讯使用 UDP 协议传输数据，**默认不加密**：

**安全特性：**

- 数据仅在局域网内传输，不经过互联网
- 不会上传到云端服务器
- 所有数据存储在本地

**安全建议：**

- 仅在可信的局域网内使用（如家庭、公司内网）
- 避免在公共 Wi-Fi 环境下使用
- 不要发送敏感信息（如密码、银行卡号）

**未来计划：**

- 考虑在后续版本中添加端到端加密（E2EE）支持

### 飞秋通讯会收集我的数据吗？

不会，飞秋通讯完全尊重您的隐私：

**数据政策：**

- ❌ 不收集任何个人信息
- ❌ 不上传聊天记录
- ❌ 不上传文件传输记录
- ❌ 不包含用户行为分析
- ❌ 不包含广告

**数据存储：**

- 所有数据存储在您的本地设备上
- 聊天记录保存在 SQLite 数据库中
- 接收的文件保存在您选择的目录

### 可以被他人拦截我的消息吗？

在局域网环境中，理论上网络管理员可以使用抓包工具查看 UDP 数据包：

**风险评估：**

- 家庭网络：风险低（只有家庭成员）
- 小型办公室：风险中等（IT 管理员可能监控）
- 公共网络：风险高（建议不要使用）

**保护措施：**

- 仅在可信网络中使用
- 避免发送敏感信息
- 未来版本将添加加密功能

### 可以设置密码保护吗？

目前飞秋通讯不支持密码保护功能。

**原因：**

- 飞秋通讯设计用于可信局域网环境
- 使用系统级别的用户账户权限
- 任何人能访问您的计算机都能查看数据

**替代方案：**

- 使用操作系统的用户账户和密码
- 启用操作系统级磁盘加密（如 BitLocker、FileVault）
- 锁定计算机屏幕

## 功能使用 (Feature Usage)

### 如何传输文件？

发送文件：

1. 在聊天窗口中点击"发送文件"按钮
2. 选择要发送的文件（支持多选）
3. 等待对方接受文件请求
4. 文件开始传输

接收文件：

1. 收到文件传输请求通知
2. 点击"接受"按钮
3. 选择文件保存位置
4. 等待文件传输完成

**文件传输特点：**

- 点对点传输，无需服务器中转
- 支持大文件传输
- 支持断点续传
- 可同时传输多个文件

### 可以创建群组吗？

是的，飞秋通讯支持群组功能：

**创建群组：**

1. 点击"创建群组"按钮
2. 输入群组名称和描述（可选）
3. 选择要添加的成员
4. 点击"创建"按钮

**群组功能：**

- 添加/移除成员
- 修改群组名称和描述
- 设置成员角色（普通成员/管理员/群主）
- 群组聊天历史

### 如何导出聊天记录？

导出聊天记录：

1. 在聊天窗口中点击"更多"菜单
2. 选择"导出聊天记录"
3. 选择导出格式（文本或 JSON）
4. 选择保存位置并确认

**导出格式：**

- **文本格式** - 易读的纯文本，适合人工查看
- **JSON 格式** - 机器可读，适合数据分析或备份

### 可以修改自己的用户名吗？

可以修改昵称，但用户 ID 是固定的：

**可修改的信息：**

- ✅ 昵称（显示给其他用户的名称）
- ✅ 头像

**不可修改的信息：**

- ❌ 用户 ID（系统自动生成的唯一标识符）

**修改步骤：**

1. 点击右上角的"设置"图标
2. 修改昵称和头像
3. 点击"保存"按钮

### 如何删除聊天记录？

删除聊天记录：

1. 打开要清理的聊天窗口
2. 点击"更多"菜单
3. 选择"清空聊天记录"
4. 确认删除

**警告：** 删除操作不可恢复，请谨慎操作。

**自动清理：**
您可以在设置中配置自动清理规则：

- 按时间清理（如保留最近 30 天）
- 按消息数量清理（如保留最近 1000 条）

### 可以同时与多个人聊天吗？

可以，飞秋通讯支持多窗口聊天：

**多聊天管理：**

- 同时打开多个聊天窗口
- 在左侧联系人列表切换聊天
- 每个聊天窗口独立显示消息历史

**系统通知：**

- 收到新消息时弹出通知
- 通知显示发送者和消息预览
- 可以在设置中关闭通知
